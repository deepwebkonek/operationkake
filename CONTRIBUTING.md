# Contributing to Operation Kake

Thank you for supporting **Operation Kake** and the **DWK Threat Monitoring Center**.

To contribute data or improvements, follow the guidelines below.

---

## 1. How to Submit a Domain

Create an Issue with:
- Domain name (non-clickable, plain text)
- Source/proof (screenshot, article, advisory)
- Reason for reporting
- Notes or context

DWK moderators will review each submission.

---

## 2. Accepted Data Types
We accept:
- Suspicious gambling domains
- Confirmed illegal gambling sites
- Mirror domains
- Redirecting domains
- OSINT-based observations

We **do not** accept:
- Personal data  
- Internal system data  
- Credentials  
- Private evidence  
- Hacks or exploits  

---

## 3. Classification
Domains are tagged under:
- `Reported`
- `Verified`
- `Under Investigation`

DWK moderators will assign the final classification.

---

## 4. Submitting Corrections
If you find:
- Inaccurate classifications  
- Duplicate domains  
- Dead or inactive sites  

Open an Issue or pull request.

---

## 5. Code of Conduct
Contributors must:
- Act ethically  
- Avoid accessing illegal websites  
- Respect privacy  
- Avoid harmful behavior  
- Follow applicable laws  

Abusive or malicious submissions will be rejected.
